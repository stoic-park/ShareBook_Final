## 프로젝트 소개

" 우리 집에 한 번 읽고, 읽지 않는 책들을 가까운 이웃들과 공유하는 건 어떨까?" 

라는 생각에서 시작된 쉐어북은

우리동네  중고책 대여 서비스 입니다.

한번 읽고 책장에 꽂아 놓은 책, 버리기엔 아까운 책들을 필요한 사람들과 공유하는 플랫폼으로

새로운 도서 분야의 아나바다 운동이 될 것입니다.

이번 기회에 잠자고 있는 여러분의 책들을 꺼내 보면 어떨까요?
